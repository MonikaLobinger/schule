# 2025-12-14
## Wie mappe ich stutiis und Werkstatt
Werkstatt soll nicht auf der Site veröffentlicht werden, aber auf localhost möchte ich es haben.
## Wie es läuft
### In Obsidian
#### Privat
mit Script .makepriv
##### Aus Public
- .ispub wird gelöscht
- stutiis (was Werkstatt war) wird zu .stutiis
- .Werkstatt wird zu Werkstatt
- in foty.js werden in foldertype werkstatt public auskommentiert und private ein
- damit wird die cssclass for neue Werkstatt Einträge `studies`
- .ispriv wird erzeugt
##### Aus Export
- .isexport wird gelöscht
- XXXXstutiis wird zu .stutiis
- .ispriv wird erzeugt
#### Öffentlich
mit Script .domakepub
- .ispriv wird gelöscht
- Werkstatt wird zu .Werkstatt
- .stutiis wird zu Werkstatt
- in foty.js werden in foldertype werkstatt private auskommentiert und public ein
- damit wird die cssclass for neue Werkstatt Einträge `werkstatt`
- .ispub wird erzeugt
#### Exportieren und einchecken
mit Script .prepareforexport
- .ispriv wird gelöscht (Das Script funktioniert nur wenn Privat)
- .stutiis wird zu XXXstutiis
- .isexport wird erzeugt
### Export
- In Projekte/Schule: Mit .prepareforexport Obsidian in Export Modus schalten. 
- in schoolout .makepriv aufrufen
- Aus Obsidian exportieren, Obsidian dabei im Vordergrund lassen
- Falls schoolout leer war, manchmal nochmal exportieren
- in schoolout .domakepub aufrufen
- in schoolout git add, git commit, git push
- in schoolout .makepriv aufrufen
- In Projekte/Schule: Mit .makepriv den Exportmodus aufheben
### Quellen einchecken
- Projekte/Schule muß im Export Modus sein
- in Projekte/Schule git add, git commit, git push
### Anzeige
Die Notes in .stutiis/Mitschriften erhalten cssclasses `studies`, die Notes in .Werkstatt/Mitschrifftten erhalten cssclasses `werkstatt`. `studies` zeigt eine bunten Rand an, `werkstatt` einen grauen. Ausserdem zeigt `Werkstatt` im Anzeigemodus auf dem oberen Drittel ein Wasserzeichen "Privat".
